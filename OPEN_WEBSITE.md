# Open Grind-Grades Locally

To view the Grind-Grades website in your browser:

1. Open the project folder in your file explorer.
2. Double-click `index.html`.

Alternatively, type one of these into your browser's address bar:

- `file:///workspaces/GrindGrades/index.html`
- `file:///path/to/your/workspace/GrindGrades/index.html`

If you are running this in a codespace or remote container, you can also use a local web server and open the served URL.

## Local server example

From the `GrindGrades` folder, run:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
