# GrindGrades.com Study Planner

A lightweight student athlete planning app for balancing academics, training, recruiting, and long-term goals.

## Features

- Grade Tracker: Monitor assignments, GPA, and academic progress.
- Training Planner: Organize workouts, practices, and recovery schedules.
- Recruiting Hub: Store stats, highlight videos, and coach contacts.
- Study Planner: Log sessions by subject, set goals, and use Pomodoro focus cycles.
- Analytics: See total study hours, best subject, streaks, and progress charts.

## Getting Started

Open `index.html` in a browser to run the app locally.

## Contact

Email: info@grindgrades.com

## Usage

1. Add a subject using the Subjects form.
2. Select a subject and click `Start Study Session`.
3. Click `End Session` when done.
4. Add study goals with a target number of hours.
5. Use the Pomodoro timer to stay focused.

Your data is saved locally in the browser using `localStorage`.
